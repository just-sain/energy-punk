# energy-punk
Игрок находит рабочий генератор в разрушенном районе города и постепенно восстанавливает поселение выживших вокруг него. Генератор — сердце базы: без энергии останавливаются все системы. Ресурсы в окружающих зданиях ограничены, поэтому игрок вынужден отправлять добровольцев в опасные рейды. Чем активнее действия игрока, тем выше угроза зомби-атак на базу. Цель — как можно дольше поддерживать жизнь поселения, балансируя между развитием, риском и выживанием.
