# energy-punk
Игрок находит рабочий генератор в разрушенном районе города и постепенно восстанавливает поселение выживших вокруг него. Генератор — сердце базы: без энергии останавливаются все системы. Ресурсы в окружающих зданиях ограничены, поэтому игрок вынужден отправлять добровольцев в опасные рейды.
